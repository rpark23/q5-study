# Lecture 01: CNS Targets

## Neurotransmission Steps

| # | Step |
|---|------|
| 1 | Stimulus triggers AP via **voltage-gated Na/K channels** |
| 2 | AP depolarizes terminal → **Ca enters via voltage-gated Ca channels** |
| 3 | Ca influx → **vesicle fusion releases NT into synaptic cleft** |
| 4 | NT **binds postsynaptic receptors** → downstream signaling |
| 5 | Termination via: **reuptake, enzymatic degradation, metabolism, diffusion** |

---

## Excitatory Neurotransmission (Glutamate)

- Main CNS excitatory NT: **Glutamate**
- Vesicular transporter: **VGLUT**
- Receptors: **AMPA** (main), NMDA, Kainate
- Signal: excitatory depolarization via cation influx
- Termination: uptake into glia via diffusion

## G-Protein Signaling

| Receptor | Coupled to | Effect |
|----------|------------|--------|
| **Gs** (B1, B2, D1) | Adenylyl cyclase | ↑ cAMP → PKA → phosphorylates substrates |
| **Gi** (M2, α2, D2) | Adenylyl cyclase | ↓ cAMP |
| **Gq** (M1, M3, α1) | Phospholipase C | PIP2 → IP3 + DAG → Ca²⁺ release → PKC |

### Gq Feedback Loops
1. High Ca²⁺ inhibits IP3 release of more Ca²⁺
2. Ca²⁺-calmodulin activates ATP pumps to lower cytosolic Ca²⁺

## Neurotransmitter Systems

### Acetylcholine (ACh)
- **Biosynthesis:** Choline Acetyltransferase (ChAT)
- **Packaging:** Vesicular Acetylcholine Transporter (VAChT)
- **Termination:** Acetylcholinesterase (AChE) in synaptic cleft
- **Receptors:** Muscarinic M1-M5 (GPCR), nicotinic (cation channels)
- **Function:** Memory

### Histamine
- **Biosynthesis:** Histidine decarboxylase
- **Packaging:** VMAT1/2
- **Termination:** Glia uptake, MAO
- **Receptors:** H1-H3 (GPCR)
- **Function:** Sleep/wake

### Norepinephrine (NE)
- **Biosynthesis:** Phenylalanine/Tyrosine → L-Dopa → Dopamine → NE (methylation → Epinephrine)
- **Packaging:** VMAT1/2
- **Termination:** NET reuptake, MAO, COMT
- **Receptors:** α1, α2, β1, β2 (GPCRs)
- **Function:** Arousal/Attention

### Dopamine (DA)
- **Biosynthesis:** Phenylalanine/Tyrosine → L-Dopa → Dopamine
- **Packaging:** VMAT1/2
- **Termination:** DAT reuptake, MAO, COMT
- **Receptors:** D1 family (Gs), D2 family (Gi) – GPCRs
- **Function:** Reward, Movement

### Serotonin (5-HT)
- **Biosynthesis:** Tryptophan → decarboxylase
- **Packaging:** VMAT1/2
- **Termination:** SERT reuptake, MAO
- **Receptors:** 5HT 1-7 (mostly GPCRs; 5HT3 is a cation channel)
- **Function:** Happiness, libido, autonomic GI regulation
